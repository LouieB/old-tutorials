One of the nice things about web development is that most of the tools and resources you need are freely available, but it's often difficult to know which ones to trust, and which will really be accurate and helpful. Following are the tools and resources I use regularly.

Core Tools

Our lab machines already have all the software you'll need installed, but if you want to do web development on your own computer, you'll need the following core tools:

Web-Optimized Editor

You can edit web files in any text editor, but any one of the following will make your life much easier. I use Visual Studio Code for JavaScript programming, and Sublime Text when writing longer HTML documents.

Sublime Text
Many web developers use this as their standard editor. It's very fast and approachable, but it can be customized extensively via plug-in packages. After you install Sublime, install Package Control and then use that to install plug-ins such as emmet completions. It can be used perpetually without a license, but it will show you a nag screen every once and while. As far as I know, they don't offer free student licenses.
Microsoft Visual Studio Code
A new, cross-platform, and completely free code editor from Microsoft! It comes with lots of handy features, and can be customized through a growing library of extensions. It has much better JavaScript support than Sublime, including a built-in debugger for Node.js projects. This is actually a web application, meaning that it's written using HTML, CSS, and JavaScript. Because it's not written in a compiled language, it won't be quite as snappy as Sublime, but it's surprisingly quick and functional.
Brackets
This is another popular editor from Adobe that is specifically targeted at client-side web development. It has some intriguing features that are not yet in either Sublime or Visual Studio Code.

Browser with Developer Tools

Google Chrome
Chrome features a great set of developer tools, and it's very fast.
Firefox
Firefox now has it's own built-in developer tools, or you can use the classic Firebug add-in.

Git and GitHub Account

git
Currently the most popular version control system. We will use this to track version of our files, back them up to GitHub, and deploy them to our web servers.
GitHub Account
Git is a decentralized version control system that doesn't strictly require a central server, but most teams prefer to use a cloud service to store the authoritative version of their code. GitHub is the most popular of those services, so create an account for yourself if you don't already have one.

Node.js

Node.js
Although Node was originally created for building simple network services, it has since become the foundation for a whole host of tools that are useful for client-side web development. It also includes a fabulous package manager, npm, which we will use to install various modules and tools.

Books, Tutorials, References and Tools

The following resources will also be helpful as you design and build your web sites and applications.

Free Books

Dive Into HTML 5
An introduction to the new features in HTML 5, written for those who already know the basics of HTML.
Eloquent JavaScript
A nice introduction to programming in JavaScript. It covers the language itself, using JavaScipt in the browser, and one chapter on building a web server in JavaScript using Node.js.
A Re-Introduction to JavaScript
A focused tutorial on JavaScript for those who are already comfortable with basic programming concepts.
You Don't Know JS
A series of free online books by Kyle Simpson that cover JavaScript in detail. If you're still struggling to understand closures, read his Scope and Closures book.
JavaScript for Cats
A gentler introduction to JavaScript for "Scaredy-Cats."

Books for Purchase

HTML & CSS: Design and Build Websites
A beautiful and informative book on HTML and CSS. It teaches you not only the syntax and basic mechanics, but also how to design effective web pages.
JavaScript: The Definitive Guide
Affectionately known as "the Rhino book," this really is the definitive guide to the JavaScript language and the DOM. The recent editions also include a chapter on the jQuery library. If you want to be a serious web developer, you should read this book at some point.

Deals for Students

GitHub Student Pack
A set of discounted or free services and products available to students, courtesy of GitHub.
JetBrains Free Student License
Get a free student license key for any JetBrains IDE, including IntelliJ Idea for Java development, or WebStorm for web development.

Technical References

General HTML 5 Reference
A general reference for HTML 5 elements, organized by function. See also the alphabetical reference.
Global Attributes Reference
A general reference for the global HTML attributes, which may be used on any element in the body section of your document.
HTML Character Entities
A reference for all the various supported character entities. Use these to encode reserved characters and other symbols.

CSS

CSS Selectors Reference
A handy table showing all the various kinds of CSS selectors with examples and descriptions of what they select.
CSS Properties Reference
A complete reference for all the various CSS properties, organized by function.
CSS Units Reference
Short reference of the various unit measurements supported in CSS and their suffixes.

JavaScript References

JavaScript and the DOM Reference
A general reference for the core JavaScript language and the Document Object Model (DOM).
JavaScript Style Guide
A reasonable though highly-opinionated style guide for JavaScript code.

Regular Expressions

Regular Expressions 101
A really impressive interactive tool for building, testing, and explaining regular expressions.

Browser Support

Can I Use...
A site that tracks support for different HTML, CSS, and JavaScript features across all major browsers (including mobile versions)

Validation Tools

W3C HTML Validation Service
Use this to check your HTML pages for invalid markup. Note that Angular uses many custom attributes that will be flagged as invalid by this tool, so don't use this with Angular-based pages.
W3C CSS Validation Service
Use this to validate your CSS pages. Browsers will just ignore invalid content in your CSS pages without any warnings, so if you are having troubles getting your CSS to work, try validating your stylesheet to make sure you don't have an error.
JS Lint
Use this to detect potential problems in your JavaScript. This is a very aggressive lint tool, however, so take its advice with a grain of salt. IDEs like Visual Studio Code and Brackets already have integrated linters built-in, and there are several linter packages for Sublime.

Color

Adobe Color CC
An amazing tool for generating color schemes. You can also upload a photo to build a color schema based on the photo's dominant colors.
Paletton Color Schema Designer
A helpful tool for generating color schemes from a base color
Colour Lovers
A site with tons of free color palettes and patterns.

Fonts and Text

Google Fonts
A collection of free custom fonts that you can reference directly in your stylesheets or HTML pages. Click on the 'Quick-use' button next to any font to get the necessary HTML/CSS.
Lorem Ipsum Generator
A tool for generating any amount of "lorem ipsum" text. Web developers and designers use this to add dummy text to their pages in order to see how their font and page layout choices will look. If you use an editor with emmet completions, you can also generate this by just typing lorem (all lower case) and then hit tab. To generate more words, add a number of words to the end: lorem100. For other humorous ipsum generators, see 15 Funny Lorem Ipsum Generators

Photos

Morgue File
Searchable set of free stock photos, some professional and some amateur. No attribution required. Pro tip: sort results by "most downloads" to see the best ones.
Pixabay
Searchable set of free stock photos, vector drawings, and illustrations. Most require attribution.
FreeImages
Searchable set of free and "premium" photos.
Tiny JPG/PNG
Online tool that compresses JPGs and PNGs without any noticeable loss of quality. This can reduce large image files by significant amounts, greatly reducing download time over slower mobile networks.

Icons

Font Awesome
A custom font with a bunch of icons instead of letters. Because this is a font instead of a PNG, you can scale and color the icons using the normal CSS text formatting properties. They also offer some additional CSS classes to animate, rotate, flip and stack the icons.
Flat Icon
A large set of flat-look icons that you can download individually as SVG (scalable) or PNG (fixed) images, or assemble a set into a custom font. The latter option has all the benefits of Font Awesome, but you get to choose which icons you want. Attribution is required.
Flat Icons
Customizable and free flat-look icons.
Icon Finder
A fabulous collection of icons, most of which are free and come in a variety of sizes.
The Noun Project
A collection of simple, flat, monochromatic icons. All of these require attribution.
SVG Optimizer
An online tool that will optimize SVG icons, removing unecessary elements and noise. Very handy if you want to include an SVG icon inline in your HTML.
